## Visão Geral

O componente `Breadcrumb` é uma lista de links que ajudam a visualizar a localização de uma página dentro da estrutura hierárquica de um site, permite a navegação até qualquer um das páginas anteriores.

## Como usar:

```tsx
<BreadcrumbRoot pageName="Nome da página" href="/pagina">
  <Breadcrumb pageName="Nome da página" href="/proxima-pagina" />
</BreadcrumbRoot>
```
