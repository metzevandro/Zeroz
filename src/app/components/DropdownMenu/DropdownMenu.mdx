## Visão Geral

O componente `DropdownMenu` são ativadas por um botão e mostram uma lista de opções, permitindo que os usuários selecionem uma ou mais opções. Uma opção escolhida pode levar a uma página ou ser utilizada como uma ação para filtrar ou classificar o conteúdo existente.

## Como usar:

```tsx
const [isDropDownOpen, setIsDropDownOpen] = useState(false);

const toggleDropDown = () => {
  setIsDropDownOpen(!isDropDownOpen);
};
return (
  <>
    <div>
      <Button
        size="md"
        variant="primary"
        label="Click here!"
        onClick={toggleDropDown}
      />
      <div>
        <DropDownMenu dropDownMenu={isDropDownOpen}>
          <DropDownMenuTitle content="Title" />
          <DropDownMenuItem content="Item 1" typeIcon="ac_unit" />
          <DropDownMenuItem content="Item 2" typeIcon="ac_unit" />
          <DropDownMenuItem content="Item 3" typeIcon="ac_unit" />
          <DropDownMenuTitle content="Title" />
          <DropDownMenuItem content="Item 1" />
          <DropDownMenuItem content="Item 2" />
        </DropDownMenu>
      </div>
    </div>
  </>
);
```
