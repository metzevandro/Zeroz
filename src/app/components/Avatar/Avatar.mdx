## Visão Geral

O componente `Avatar` é utilizado para utilizar como um local de imagem do usuário. Caso não haja uma imagem, como padrão virá com um `icon="person"` dentro dele. Além disso, ele possuí três tamanhos, sendo eles: `sm`, `md` e `lg`.
